## Overview

The Place Enrichment API allows you to enhance your location data with rich, structured attributes. Submit place information and receive enriched data including verified details, categorization, and custom attributes.

## Key Features

- **Flexible Input** - Accept partial or complete place information
- **AI-Powered Enrichment** - Intelligent data enhancement using multiple sources
- **Custom Attributes** - Define and extract organization-specific data points
- **Job Management** - Track enrichment status and retrieve results
- **Batch Processing** - Enrich multiple places efficiently

## Quick Start

<Steps>
  <Step title="Get Your API Key">
    Visit your [dashboard](https://app.repromptai.com) to get your organization's API key. See the [Authentication](/guides/authentication) guide for details.
  </Step>
  
  <Step title="Submit an Enrichment Request">
    Send a POST request to the enrichment endpoint with your place data:
    
    <CodeGroup>
    
    ```javascript Node.js
    const response = await fetch(
      'https://reprompt--reprompt-fastapi-fastapi-app.us-west.modal.run/{org_slug}/place_enrichment/enrich',
      {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          place_id: 'place_001',
          inputs: {
            name: "Joe's Pizza",
            latitude: 40.7359,
            longitude: -73.9911,
            address: "123 Main St, New York, NY 10001"
          }
        })
      }
    );
    
    const data = await response.json();
    console.log(data);
    ```
    
    ```python Python
    import requests
    
    response = requests.post(
        'https://reprompt--reprompt-fastapi-fastapi-app.us-west.modal.run/{org_slug}/place_enrichment/enrich',
        headers={
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
        },
        json={
            'place_id': 'place_001',
            'inputs': {
                'name': "Joe's Pizza",
                'latitude': 40.7359,
                'longitude': -73.9911,
                'address': '123 Main St, New York, NY 10001'
            }
        }
    )
    
    data = response.json()
    print(data)
    ```
    
    </CodeGroup>
  </Step>
  
  <Step title="Check Job Status">
    Use the returned job ID to check the enrichment status:
    
    ```javascript
    const jobResponse = await fetch(
      'https://reprompt--reprompt-fastapi-fastapi-app.us-west.modal.run/{org_slug}/place_enrichment/jobs/place_001',
      {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY'
        }
      }
    );
    
    const job = await jobResponse.json();
    console.log(job.status); // pending, processing, completed, or failed
    ```
  </Step>
  
  <Step title="Retrieve Enriched Data">
    Once the job is complete, the response includes all enriched attributes.
  </Step>
</Steps>

## Input Requirements

### Required Fields
- **place_id**: Your unique identifier for this place

### Optional Input Fields
- **name**: Place name
- **address**: Full address or address components
- **latitude/longitude**: Geographic coordinates
- **category**: Place category or type
- **phone**: Phone number
- **website**: Website URL

<Note>
  The more information you provide, the more accurate the enrichment results will be. However, the API can work with minimal input.
</Note>

## Available Attributes

The Place Enrichment API can extract and verify numerous attributes:

- **Basic Information**: Name, address, phone, website
- **Location Data**: Coordinates, timezone, postal code
- **Business Details**: Hours of operation, price range, ratings
- **Categories**: Primary and secondary categorization
- **Custom Attributes**: Organization-defined fields

To see all available attributes, use the `/place_enrichment/attributes` endpoint.

## Job Lifecycle

1. **Submitted** - Enrichment request received
2. **Processing** - AI extraction and verification in progress
3. **Completed** - Enriched data ready
4. **Failed** - Error occurred (check error message)

## Error Handling

<CodeGroup>

```javascript Node.js
try {
  const response = await fetch(url, options);
  
  if (!response.ok) {
    const error = await response.json();
    console.error('Enrichment failed:', error.detail);
    throw new Error(error.detail);
  }
  
  const data = await response.json();
  return data;
} catch (error) {
  console.error('Request failed:', error);
}
```

```python Python
try:
    response = requests.post(url, headers=headers, json=payload)

    print(f'Request failed: {e}')
```
