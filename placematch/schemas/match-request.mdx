# Match Request Schema

The request schema for the Place Match API endpoints.

## MatchRequest

```json
{
  "source_place": {
    "$ref": "#/components/schemas/PlaceInput"
  },
  "candidate_places": [
    {
      "$ref": "#/components/schemas/CandidatePlace"
    }
  ],
  "options": {
    "$ref": "#/components/schemas/MatchOptions"
  }
}
```

## PlaceInput

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Business or place name |
| `address` | string | Yes | Full address string |
| `latitude` | number | Yes | Latitude coordinate (-90 to 90) |
| `longitude` | number | Yes | Longitude coordinate (-180 to 180) |
| `phone` | string | No | Phone number in any format |
| `website` | string | No | Website URL |
| `category` | string | No | Business category or type |

```json
{
  "name": "Joe's Pizza",
  "address": "123 Main St, New York, NY 10001",
  "latitude": 40.7128,
  "longitude": -74.0060,
  "phone": "+1-212-555-0123",
  "website": "https://joespizza.com",
  "category": "restaurant"
}
```

## CandidatePlace

Extends `PlaceInput` with an additional `id` field:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | Yes | Unique identifier for the candidate |
| All fields from `PlaceInput` | | | |

```json
{
  "id": "candidate_123",
  "name": "Joe's Pizza Restaurant",
  "address": "123 Main Street, New York, NY 10001",
  "latitude": 40.7129,
  "longitude": -74.0061,
  "phone": "212-555-0123",
  "website": "https://www.joespizza.com"
}
```

## MatchOptions

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `min_confidence` | number | 0.7 | Minimum confidence score (0.0-1.0) |
| `max_distance_meters` | number | 1000 | Maximum distance for geographic matching |
| `include_reasons` | boolean | true | Include detailed match reasoning |
| `fuzzy_name_matching` | boolean | true | Enable fuzzy string matching for names |
| `normalize_addresses` | boolean | true | Normalize addresses before comparison |

```json
{
  "min_confidence": 0.8,
  "max_distance_meters": 500,
  "include_reasons": true,
  "fuzzy_name_matching": true,
  "normalize_addresses": true
}
```

## Validation Rules

### Coordinates
- `latitude`: Must be between -90 and 90
- `longitude`: Must be between -180 and 180

### Confidence Threshold
- `min_confidence`: Must be between 0.0 and 1.0

### Distance
- `max_distance_meters`: Must be positive integer, max 50,000 (50km)

### Limits
- `candidate_places`: Maximum 100 candidates per request
- `name`: Maximum 200 characters
- `address`: Maximum 500 characters
