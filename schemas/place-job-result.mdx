# PlaceJobResult

The `PlaceJobResult` model represents the result of a completed place enrichment job, containing both the original input data and the enriched attributes.

## Schema Definition

<Note>
This schema documentation is manually maintained but represents the structure defined in your OpenAPI specification. Keep this in sync with your Pydantic models.
</Note>

<ResponseField name="job_id" type="string" required>
  Unique identifier for the job.
  
  **Example:** `"job_abc123def456"`
</ResponseField>

<ResponseField name="place_id" type="string" required>
  The customer-provided place identifier.
  
  **Example:** `"place_nyc_123"`
</ResponseField>

<ResponseField name="status" type="string" required>
  Current status of the job.
  
  **Options:** `"pending"` | `"in_progress"` | `"completed"` | `"failed"`
</ResponseField>

<ResponseField name="inputs" type="UniversalPlace" required>
  The original input data provided for enrichment. See [UniversalPlace](/schemas/universal-place) for schema details.
</ResponseField>

<ResponseField name="enriched_data" type="object" optional>
  The enriched place data containing discovered attributes.
  
  **Structure varies based on requested attributes**
</ResponseField>

<ResponseField name="confidence_scores" type="object" optional>
  Confidence scores for each enriched attribute.
  
  **Example:** `{"website": 0.95, "phone": 0.87}`
</ResponseField>

<ResponseField name="created_at" type="string (datetime)" required>
  Timestamp when the job was created.
  
  **Example:** `"2024-03-15T10:30:00Z"`
</ResponseField>

<ResponseField name="completed_at" type="string (datetime)" optional>
  Timestamp when the job was completed (if applicable).
  
  **Example:** `"2024-03-15T10:32:15Z"`
</ResponseField>

<ResponseField name="error_message" type="string" optional>
  Error message if the job failed.
</ResponseField>

## Usage Examples

### Successful Job Result
```json
{
  "job_id": "job_abc123def456",
  "place_id": "place_nyc_123",
  "status": "completed",
  "inputs": {
    "name": "Joe's Pizza",
    "latitude": 40.7359,
    "longitude": -73.9911,
    "full_address": "7 Carmine St, New York, NY 10014"
  },
  "enriched_data": {
    "website": "https://joespizza.com",
    "phone": "+1-212-555-0123",
    "categories": ["restaurant", "pizza", "italian"],
    "opening_hours": {
      "monday": "11:00-22:00",
      "tuesday": "11:00-22:00"
    }
  },
  "confidence_scores": {
    "website": 0.95,
    "phone": 0.87,
    "categories": 0.92
  },
  "created_at": "2024-03-15T10:30:00Z",
  "completed_at": "2024-03-15T10:32:15Z"
}
```

### Failed Job Result
```json
{
  "job_id": "job_xyz789abc123",
  "place_id": "place_invalid_001",
  "status": "failed",
  "inputs": {
    "name": "Unknown Place"
  },
  "error_message": "Insufficient location data provided",
  "created_at": "2024-03-15T10:30:00Z"
}
```
