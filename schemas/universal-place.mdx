# UniversalPlace

The `UniversalPlace` model represents the input data structure for places to be enriched. This is the core data model used across all place enrichment operations.

## Schema Definition

<Note>
This schema documentation is manually maintained but represents the structure defined in your OpenAPI specification. Keep this in sync with your Pydantic models.
</Note>

<ResponseField name="name" type="string" optional>
  The name of the place or business.
  
  **Example:** `"Joe's Pizza"`
</ResponseField>

<ResponseField name="latitude" type="number" optional>
  Latitude coordinate of the place location.
  
  **Example:** `40.7359`
</ResponseField>

<ResponseField name="longitude" type="number" optional>
  Longitude coordinate of the place location.
  
  **Example:** `-73.9911`
</ResponseField>

<ResponseField name="full_address" type="string" optional>
  Complete address of the place.
  
  **Example:** `"7 Carmine St, New York, NY 10014"`
</ResponseField>

<ResponseField name="street_address" type="string" optional>
  Street address component only.
  
  **Example:** `"7 Carmine St"`
</ResponseField>

<ResponseField name="city" type="string" optional>
  City name.
  
  **Example:** `"New York"`
</ResponseField>

<ResponseField name="state" type="string" optional>
  State or province.
  
  **Example:** `"NY"`
</ResponseField>

<ResponseField name="postal_code" type="string" optional>
  ZIP or postal code.
  
  **Example:** `"10014"`
</ResponseField>

<ResponseField name="country" type="string" optional>
  Country name or code.
  
  **Example:** `"US"`
</ResponseField>

<ResponseField name="phone" type="string" optional>
  Phone number of the place.
  
  **Example:** `"+1-212-555-0123"`
</ResponseField>

<ResponseField name="website" type="string" optional>
  Website URL of the place.
  
  **Example:** `"https://joespizza.com"`
</ResponseField>

## Usage Examples

### Minimal Place Data
```json
{
  "name": "Central Coffee",
  "latitude": 40.7614,
  "longitude": -73.9776
}
```

### Complete Place Data
```json
{
  "name": "The Italian Corner",
  "latitude": 40.7589,
  "longitude": -73.9851,
  "full_address": "123 Main St, New York, NY 10001",
  "street_address": "123 Main St",
  "city": "New York",
  "state": "NY",
  "postal_code": "10001",
  "country": "US",
  "phone": "+1-212-555-0199",
  "website": "https://italiancorner.com"
}
```
